# 2D-Minecraft-Clone

Basic 2D block game using C++ and OpenGL, that incorperates perlin noise terrain generation, physics and block placement. Also build in python.


## Features

* Perlin Noise Terrain Generation
* Player physics
* Screen shifts to player
* Block remove/place
